Image Generator MCP Server

This is an MCP (Model Context Protocol) server that generates images from text prompts using Google Imagen.

Installation: 
claude mcp add image-generator https://image-generator-mcp.netlify.app/mcp

Configuration

You'll need to configure your Google API key to use this service. Get your API key from the Google AI Studio.

Available Tools

  • generate_image - Generate an image from a text prompt
    • prompt (required): The text description of the image to generate
    • negativePrompt (optional): What to avoid in the generated image
    • aspectRatio (optional): Image aspect ratio (1:1, 16:9, 9:16, 4:3, 3:4)

Example Usage

After installation, you can use prompts like:

  • "Generate an image of a sunset over mountains"
  • "Create an image of a futuristic city with flying cars"
  • "Make an image of a cute robot playing with a cat"

Error Handling

The server provides detailed error messages for common issues:

  • Invalid API key
  • API quota exceeded
  • Safety filters triggered
  • General generation errors
